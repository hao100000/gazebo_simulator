## gazebo_simulator/models/モデル名/urdf/モデル名.xacro
### 6,7行目あたり
```xml
<xacro:include filename="$(find arm_2)/urdf/materials.xacro" />
<xacro:include filename="$(find arm_2)/urdf/arm_2.trans" />
```
みたいなやつ(モデル名によって微妙にちがう)を
```xml
<xacro:property name="robot_name" value="$(arg robot_name)"/>

<xacro:include filename="$(find gazebo_simulator)/models/${robot_name}/urdf/materials.xacro" />
<xacro:include filename="$(find gazebo_simulator)/models/${robot_name}/urdf/${robot_name}.trans" />
```
に変更

### 末尾の</robot>の直前
以下を追加。
```xml
<!-- ros2_control plugin config (keep velocity interfaces compatible with existing controllers) -->
<ros2_control name="GazeboSystem" type="system">
  <hardware>
    <plugin>gz_ros2_control/GazeboSimSystem</plugin>
  </hardware>

  <joint name="motor_1">
    <command_interface name="velocity"/>
    <state_interface name="position"/>
    <state_interface name="velocity"/>
  </joint>

  <joint name="motor_2">
    <command_interface name="velocity"/>
    <state_interface name="position"/>
    <state_interface name="velocity"/>
  </joint>  

</ros2_control>

<gazebo>
  <plugin filename="libgz_ros2_control-system.so" name="gz_ros2_control::GazeboSimROS2ControlPlugin">
    <parameters>$(find gazebo_simulator)/config/controller_${robot_name}.yaml</parameters>
  </plugin>
</gazebo>

<gazebo>
  <plugin
    filename="libgz-sim-pose-publisher-system.so"
    name="gz::sim::systems::PosePublisher">
    <publish_link_pose>true</publish_link_pose>
    <publish_model_pose>true</publish_model_pose>
    <use_pose_vector_msg>true</use_pose_vector_msg>
    <update_frequency>50</update_frequency>
  </plugin>
</gazebo>
```
この際、<joint>がジョイントに対応するので、自分のモデルのジョイントに対応するように変更。  
自分のモデルのジョイントの名前は
```xml
<!-- ros2_control plugin config (keep velocity interfaces compatible with existing controllers) -->
```
のすぐ上の<joint>タグで確認できる。
```xml
  <joint name="motor_1">
    <command_interface name="velocity"/>
    <state_interface name="position"/>
    <state_interface name="velocity"/>
  </joint>
```
をいい感じにコピペして下にどんどん追記していけばOK。
